Introduction

This section covers the management of GAMS models for the WebUI on a server. Section Turn your GAMS model into a WebUI has shown what to do on the coding side to be able to use a GAMS model in the WebUI. Now we take such an adapted model and show how to make it available for a WebUI running on a server. Furthermore, it is shown how authorisation-regulated input data can be stored in the server's database to be accessable by authorized users.
