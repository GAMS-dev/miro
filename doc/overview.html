The GAMS WebUI

Last updated: Jauary 10th, 2019

Why should I use the WebUI?

Are you working with GAMS and looking for a fast way to deploy your GAMS models automatically?
Do you have to make decisions based on GAMS optimization problems and are looking for an intuitive visualization tool?

Then you should give the GAMS WebUI a chance!

  • Quick & automated deployment of GAMS models
  • Data visualization with powerful graphics
  • Sophisticated generation, processing, evaluation & storing of scenario data
  • Creation & analysis of performance statistics
  • Analysis of the parameter range of a model
  • Intuitive planning without GAMS knowledge
  • Easy to use & comfortable working environment

We have developed this tool to extend the possibilities to work with GAMS models.

Run the GAMS WebUI on a server

You can also run the GAMS WebUI on a server. This adds multi-user and multi-application support as well as many enterprise features such as authorisation via e.g. LDAP, load balancing, rolling updates and much more. Interested? Please contact us!
