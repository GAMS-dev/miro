Installation

Requirements

GAMS

To be able to use GAMS MIRO for your model, a current GAMS version (NOTE: a version is not specified until the software is released.) is required. You can download GAMS here. You will also find installation instructions for the GAMS system there.

R

A current R version (3.5.1 or later) is required for using MIRO. You can download it here.

  • Windows
  • MacOS
  • Linux

GAMS MIRO installation

To install MIRO and its components, download the latest version and run the setup, which will guide you through the installation process.
The installer checks for an existing R version on your system and installs a current one, if necessary.

Demo applications

There are demo applications that can help you get started with MIRO and its functionality. Those demo applications are located in C:\Users\<user_Name>\MIRO_examples\ after a successful installation. In case you want to download them again, you can find the demo applications here. To use them, simply unzip the downloaded file and move the folder into a directory of your choice.

GAMS MIRO installation

To install MIRO and its components, download the latest version.
