Installation

Last updated: Jauary 11th, 2019

Requirements

GAMS

To be able to use the GAMS WebUI for your model, a current GAMS version (25.1.2 or later) is required. You can download it here. You will also find installation instructions for the GAMS system there.

R

A current R version (3.5.1 or later) is required for using the WebUI. You can download it here.

Windows

GAMS WebUI Installation

To install the GAMS WebUI and its components, download the latest WebUI version and run the setup, which will guide you through the installation process.
The installer checks for an existing R version on your system and installs a current one, if necessary.

Demo applications

There are demo applications that can help you get started with the WebUI and its functionality. Those demo applications are located in C:\Users\<user_Name>\MIRO_examples\ after a successful installation. In case you want to download them again, you can find the demo applications here [link missing]. To use it, simply unzip the downloaded file and move the folder into a directory of your choice.
