GAMS Installation

To be able to use the GAMS WebUI for your model, a current GAMS version (25.1.2 or later) is required. You can download it here. Under the given link you will also find installation instructions for the GAMS system.

R and GAMS WebUI Installation

To install the GAMS WebUI and its components, download the latest WebUI version and run the setup, which will guide you through the installation process.
To be able to use the application, a current R version (3.5.1 or later) is required. If you install the WebUI on a Windows computer, the installer checks for an existing R version on your system and installs it, if necessary.
Note: On Mac and Linux, you currently need to install R manually (download here) in case it is not yet installed on your system.

Demo applications

There are demo applications that can help you get started with the WebUI and its functionality. On windows those demo applications are located in C:\Users\<user_Name>\GMSWebUI\examples\ after a successful installation. If you are using MacOS or Linux, you have to download the applications first. You can find them here [link missing]. To use it, simply unzip the downloaded file and move the folder into a directory of your choice.
